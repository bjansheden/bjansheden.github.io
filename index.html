Sound the Alarm!

"Sound the Alarm!" is an interactive sonification of Covid-19's spread in Sweden spanning from February 27th 2020 until February 27th 2021. The project was written in Pure Data and the user interface was also created in Pure Data.

  • Watch the demo

SLAR

"SLAR" is an AR-map used to provide the user with a new perspective on the frequently used SL map. By visualizing a 2D-map in a 3D-environment the user gains the ability to tilt the map and view it from the preffered position. The models of the map was created in Unity and the AR-functionality uses the Vuforia Engine.
